# JobLink - 智能求职助手

JobLink 是一个现代化的求职平台，帮助用户找到理想的工作机会。

## 功能特点

- 智能职位匹配
- 实时职位更新
- 个性化求职体验
- 公司详情展示
- 职位分类浏览

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/JobLink.git
```

2. 安装依赖
```bash
cd JobLink
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 打开 [http://localhost:3000](http://localhost:3000) 查看效果

## 部署

本项目使用 Vercel 进行部署。每次推送到 main 分支时，Vercel 会自动部署最新版本。
